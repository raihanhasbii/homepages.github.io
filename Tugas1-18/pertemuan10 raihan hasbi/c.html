1. Apa gunanya IFRAME?

IFRAME (Inline Frame) adalah elemen HTML yang digunakan untuk menampilkan halaman web atau dokumen HTML lain di dalam halaman web utama. IFRAME berguna untuk mengintegrasikan atau menanamkan konten dari sumber eksternal ke dalam halaman web Anda. Misalnya, Anda dapat menggunakan IFRAME untuk menampilkan video YouTube, peta Google Maps, atau halaman web eksternal lainnya di dalam halaman web Anda.

2. Apa bedanya FRAME dan IFRAME?

Perbedaan utama antara FRAME dan IFRAME adalah sebagai berikut: - FRAME: FRAME digunakan dalam HTML versi lama (HTML 4 dan sebelumnya) untuk membagi sebuah halaman web menjadi beberapa frame atau jendela terpisah. FRAME memungkinkan Anda untuk menampilkan beberapa dokumen HTML dalam satu halaman web, dan setiap frame dapat memiliki konten yang berbeda. Namun, penggunaan FRAME telah dianggap usang, dan IFRAME lebih disarankan untuk penggunaan modern. - IFRAME: IFRAME digunakan dalam HTML5 dan versi HTML yang lebih baru untuk menanamkan halaman web atau konten eksternal di dalam halaman web utama. IFRAME bekerja dalam satu frame saja dan dapat menampilkan konten eksternal, seperti video, peta, atau halaman web lainnya, secara terintegrasi dalam halaman web.

3. Tuliskan bentuk umum perintah IFRAME secara lengkap?

Perintah IFRAME memiliki bentuk umum sebagai berikut: html Penjelasan atribut-atributnya: - `src`: Atribut ini digunakan untuk menentukan URL sumber eksternal yang akan ditampilkan dalam IFRAME. - `width` dan `height`: Atribut-atribut ini digunakan untuk mengatur lebar dan tinggi IFRAME. - `frameborder`: Atribut ini digunakan untuk mengontrol tampilan bingkai (frame) IFRAME. Biasanya diatur ke "0" untuk menghilangkan bingkai. - `scrolling`: Atribut ini mengontrol apakah IFRAME harus memiliki bilah geser. Nilainya bisa "auto" (secara otomatis menampilkan bilah geser jika diperlukan) atau "no" (tanpa bilah geser).
